# Project Overview

## Name

**E-NEST - Engineers Nest**

## Tagline

A Smart Room & Mess Discovery Platform

## Objective

E-NEST is designed to help college students compare nearby accommodation and mess options through a structured discovery experience. The prototype demonstrates how students can search, filter, view details, compare locations, and manage saved listings or inquiries.

## Target Users

- College students looking for rooms, PGs, hostels, or mess plans
- Property owners who want to list rooms
- Mess operators who want to show meal plan details
- Faculty evaluators reviewing the project concept, UI flow, and planned architecture

## Current Prototype

The current version is a React-based frontend prototype with realistic placeholder data. It focuses on UI/UX, user flow, information architecture, dashboards, and implementation planning.

## Not Yet Implemented

- Live backend APIs
- Real MongoDB connection
- Real Google Maps API integration
- User authentication sessions
- Verified owner workflow
- Online booking or payment

## Planned Architecture

- React.js frontend
- Node.js and Express.js backend
- MongoDB database
- Google Maps API for maps and distance features

## Presentation Purpose

This project is prepared as a professional final-year college project showcase. It is suitable for explaining the product idea, prototype scope, planned implementation, and future roadmap.
