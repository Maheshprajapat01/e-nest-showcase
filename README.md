# E-NEST - Engineers Nest

E-NEST is a design/prototype showcase for a college final-year project:

**A Smart Room & Mess Discovery Platform**

The prototype presents a startup-quality product overview for helping college students discover nearby rooms, PGs, hostels, and mess plans using realistic placeholder data.

## Project Status

This repository is currently a **frontend design prototype**. It does not include a connected backend, database, live Google Maps integration, payment system, or real property verification.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
